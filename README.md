# phpgo
PHP有很多方便的函数，GO里面没有或者命名和PHP不一样，为了方便PHP转GO的同学方便，特写了本项目。
> 项目里面的所有函数命名都是和PHP函数一致，仅仅是首字母大写，下划线_去掉。仅支持go mod
# 使用方法
```go
import "github.com/lewjian/phpgo"

phpgo.InArray("a", []string{"a"})
```
